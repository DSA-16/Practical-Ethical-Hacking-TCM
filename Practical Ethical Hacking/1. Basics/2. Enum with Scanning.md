6. To see any file shares mounted on we use
-> showmount -e 192.168.1.15
result maybe : /srv/nfs 172.168.1.15....

to mount to that share,
first amke a directory
-> mkdir /mnt/dev
-> mount -t nfs 192.168.1.15:/srv/nfs /mnt/dev
and open the folder: /mnt/dev

7.  DNS recon:
dnsrecon -r 127.0.0.0/24 -n 192,168.1.15 -d anything
r: our range
n: target ip
d: anything (domian)
